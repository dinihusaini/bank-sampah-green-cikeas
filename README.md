# Bank Sampah Green Cikeas

Sistem Informasi Bank Sampah Green Cikeas berbasis website.

## Fitur

### Admin
- Dashboard admin
- Kelola nasabah
- Kelola jenis sampah
- Penimbangan sampah
- Tabungan sampah
- Absensi QR Code
- Laporan transaksi

### Nasabah
- Dashboard nasabah
- QR Code absensi
- Riwayat tabungan
- Informasi saldo

## Teknologi

- PHP Native
- MySQL
- Bootstrap
- JavaScript
- PHPMailer

## Database

Import file database yang tersedia pada folder:

database/

## Konfigurasi

Copy:

config/koneksi-example.php

menjadi:

config/koneksi.php

lalu sesuaikan koneksi database.
